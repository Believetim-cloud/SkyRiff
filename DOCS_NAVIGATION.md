## 🚀 如何使用

### 在Figma Make中查看

**方法A: 使用文件浏览器**
1. 在Figma Make界面的**左侧**找到文件浏览器
2. 查找以 `.md` 结尾的文件
3. 点击文件名打开查看

**方法B: 使用搜索功能**
1. 按 `Ctrl + P` (Windows) 或 `Cmd + P` (Mac)
2. 输入文件名，如 `README`
3. 回车打开

---

## 💾 下载到本地

### 方法1: 手动复制
1. 在Figma Make中打开文档文件
2. 全选内容 (`Ctrl+A` / `Cmd+A`)
3. 复制 (`Ctrl+C` / `Cmd+C`)
4. 粘贴到本地文本编辑器保存

### 方法2: 导出整个项目
1. 查找Figma Make的**导出/下载**按钮
2. 选择下载整个项目
3. 解压后即可看到所有文件

## ❓ 找不到文件？

如果您在Figma Make界面中找不到这些文档文件，可能的原因：

1. **文件在折叠的文件夹中**
   - 展开根目录查看所有文件

2. **需要刷新界面**
   - 刷新浏览器页面

3. **文档在其他标签页**
   - 检查是否有其他打开的标签

4. **界面布局问题**
   - 尝试调整窗口大小
   - 查找"显示文件浏览器"按钮
