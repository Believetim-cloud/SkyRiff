## 🎯 三种获取方式

### 方式1: 在Figma Make中直接查看 ⭐推荐

**步骤**:
1. 在Figma Make界面左侧找到文件浏览器
2. 点击根目录 `/` 
3. 找到以 `.md` 结尾的文件
4. 点击即可查看

**文件位置**:
```
/ (根目录)
├── README.md
├── COMPLETE_SOURCE_CODE.md
├── PROJECT_CODE_EXPORT.md
├── ASSETS_COMPLETE_LIST.md
└── DOCS_NAVIGATION.md
```

---

### 方式2: 手动复制到本地

#### 步骤A: 获取README.md
1. 在Figma Make中打开 `/README.md`
2. 全选内容 (Ctrl+A 或 Cmd+A)
3. 复制 (Ctrl+C 或 Cmd+C)
4. 在本地创建 `README.md` 文件
5. 粘贴保存

#### 步骤B: 重复以上步骤获取其他文档
- `COMPLETE_SOURCE_CODE.md`
- `PROJECT_CODE_EXPORT.md`
- `ASSETS_COMPLETE_LIST.md`
- `DOCS_NAVIGATION.md`

---

### 方式3: 导出整个项目 ⭐最简单

#### 如果Figma Make支持导出功能:

**查找导出按钮**:
- 在界面顶部或右上角
- 可能叫 "Export" / "Download" / "导出" / "下载"
- 图标可能是 📥 或 ⬇️

**导出步骤**:
1. 点击 Export/Download 按钮
2. 选择 "Download as ZIP" 或类似选项
3. 等待下载完成
4. 解压ZIP文件
5. 所有文档都在根目录

## 🎁 附赠：一键获取脚本

### Windows (PowerShell)

创建 `download-docs.ps1`:

```powershell
# 创建文档文件夹
New-Item -ItemType Directory -Force -Path "SkyRiff-Docs"
cd SkyRiff-Docs

# 创建空文档文件
@('README.md', 'COMPLETE_SOURCE_CODE.md', 'PROJECT_CODE_EXPORT.md', 'ASSETS_COMPLETE_LIST.md', 'DOCS_NAVIGATION.md') | ForEach-Object {
    New-Item -ItemType File -Force -Path $_
}

Write-Host "✅ 文档文件已创建在 SkyRiff-Docs 文件夹"
Write-Host "请手动复制内容到对应文件"
```

运行: `powershell -ExecutionPolicy Bypass -File download-docs.ps1`

---

### macOS/Linux (Bash)

创建 `download-docs.sh`:

```bash
#!/bin/bash

# 创建文档文件夹
mkdir -p SkyRiff-Docs
cd SkyRiff-Docs

# 创建空文档文件
touch README.md
touch COMPLETE_SOURCE_CODE.md
touch PROJECT_CODE_EXPORT.md
touch ASSETS_COMPLETE_LIST.md
touch DOCS_NAVIGATION.md

echo "✅ 文档文件已创建在 SkyRiff-Docs 文件夹"
echo "请手动复制内容到对应文件"
```

运行: `chmod +x download-docs.sh && ./download-docs.sh`
