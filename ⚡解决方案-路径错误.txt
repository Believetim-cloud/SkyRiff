🔍 问题原因：
─────────────────────────────────────────────────────────────────

您直接在 C:\Users\k> 目录运行了命令，
但 server 文件夹在项目目录下，不在 C:\Users\k> 下。


✅ 解决方案（3种方法，任选1种）：
─────────────────────────────────────────────────────────────────

═══════════════════════════════════════════════════════════════
方法1：Shift+右键（最简单）⭐⭐⭐
═══════════════════════════════════════════════════════════════

步骤：
1. 打开文件管理器
2. 找到您的 SkyRiff 项目文件夹
3. 按住 Shift 键 + 右键点击文件夹
4. 选择"在此处打开命令窗口"
5. 直接执行：
   
   cd server && npm install && npm start

✅ 完成！


═══════════════════════════════════════════════════════════════
方法2：手动 cd 进入项目（100%成功）⭐⭐
═══════════════════════════════════════════════════════════════

步骤：
1. 打开 cmd（Win+R → 输入 cmd → 回车）

2. 找到项目路径（例如：C:\Users\k\Desktop\skyriff）

3. 输入以下命令（把路径改成您的实际路径）：

   cd C:\Users\k\Desktop\skyriff
   cd server
   npm install
   npm start

✅ 完成！


═══════════════════════════════════════════════════════════════
方法3：一条命令搞定⭐
═══════════════════════════════════════════════════════════════

把下面的路径改成您的项目实际路径，然后复制粘贴到 cmd：

cd C:\Users\k\Desktop\skyriff && cd server && npm install && npm start

（把 C:\Users\k\Desktop\skyriff 改成您的项目路径）

✅ 完成！


═══════════════════════════════════════════════════════════════


📝 如何找到项目路径？
─────────────────────────────────────────────────────────────────

方法1：
  1. 在文件管理器中打开项目文件夹
  2. 点击地址栏（顶部的路径）
  3. 地址栏会变成文本，全选复制即可

方法2：
  1. 右键项目文件夹 → 属性
  2. 查看"位置"字段
  3. 复制这个路径 + \skyriff
