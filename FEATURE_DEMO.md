## 1️⃣ 🎬 生成AI视频

### 功能亮点
✨ 无需任何素材，只需文字描述  
✨ 8种专业模型可选  
✨ 实时显示生成进度  
✨ 3-5分钟快速生成  

### 操作步骤

#### 方法A：通过视频工作室（推荐）

```
第1步：点击底部导航栏中间的"工作室"按钮
      （有视频图标的大圆按钮）

第2步：看到4个精美的功能卡片
      点击第一个："🎬 生成AI视频"

第3步：在弹出的对话框中
      - 确保选择"文生视频"模式（左侧按钮）
      - 在文本框输入提示词

第4步：输入提示词示例
      试试这些：
      ✅ "可爱的狗 开飞机"
      ✅ "金色日落下的海浪拍打沙滩"
      ✅ "宇宙中的星球旋转，星云闪耀"
      ✅ "森林中的小溪流水，阳光透过树叶"

第5步：选择模型
      推荐新手选择："竖屏 15秒"
      （平衡了质量和速度）

第6步：点击底部"开始生成"按钮
      - 会显示进度条
      - 预计等待3-5分钟
      - 进度从0%到100%

第7步：生成成功！
      - 看到绿色成功提示
      - 自动跳转到资产页
      - 视频已保存到本地
```

#### 方法B：通过工具页

```
第1步：点击底部"工具"Tab

第2步：点击"AI视频生成"卡片

第3步：同样会打开视频生成器
      后续步骤与方法A相同
```

### 💡 提示词技巧

**基础模板**：
```
[主体] + [动作] + [环境] + [氛围]

示例：
"一只金色的狐狸 / 在秋天的森林中奔跑 / 落叶纷飞 / 温暖的阳光"
```

**优秀示例**：
```
自然景观：
✓ "璀璨的星空夜景，流星划过天际，银河清晰可见"
✓ "清晨的薄雾笼罩着湖面，远处山峦若隐若现"
✓ "雪山之巅的日出，云海翻涌，金光洒满山顶"

动物主题：
✓ "蝴蝶在花丛中翩翩起舞，色彩斑斓的花瓣"
✓ "一群海豚在蔚蓝的海面跃出水面，溅起水花"
✓ "小鸟在树枝上欢快鸣叫，春意盎然"

城市场景：
✓ "繁华都市的霓虹灯光，车流穿梭，充满未来感"
✓ "雨夜街道的倒影，行人撑伞匆匆，浪漫氛围"
✓ "现代化城市天际线，黄昏时分，余晖洒在玻璃幕墙"

## 2️⃣ 📸 图片转视频

### 功能亮点
✨ 让静态图片动起来  
✨ 支持本地上传和URL输入  
✨ 添加各种动态效果  
✨ 风格转换和增强  

### 操作步骤

```
第1步：准备一张图片
      - 可以是风景照片
      - 可以是人物画像（非真人）
      - 可以是艺术作品

第2步：打开"工作室" → "📸 图片转视频"

第3步：上传图片（两种方式）
      方式A：点击"选择图片文件"
            - 从电脑选择图片
            - 自动显示预览
      
      方式B：输入图片URL
            - 粘贴在线图片链接
            - 按回车加载预览

第4步：输入提示词
      描述你想要的效果：
      
      基础效果：
      ✓ "让画面动起来"
      ✓ "添加动态效果"
      
      添加元素：
      ✓ "添加飘落的樱花花瓣"
      ✓ "添加下雨效果"
      ✓ "添加粒子光效"
      
      风格转换：
      ✓ "改成电影风格，增强色彩"
      ✓ "变成动画风格"
      ✓ "转换为水彩画风格"
      
      增强画面：
      ✓ "增强光影对比，添加景深"
      ✓ "添加镜头光晕，电影级画质"

第5步：选择模型，点击"开始生成"

第6步：等待完成（3-5分钟）

第7步：查看结果
      - 静态图片变成了动态视频
      - 可以播放查看效果
```

### 💡 图片选择建议

**适合的图片**：
```
✅ 风景照片（山川、湖泊、城市）
✅ 艺术作品（插画、绘画）
✅ 抽象图案（几何、色彩）
✅ 静物摄影（花卉、物品）
✅ 建筑摄影（古建筑、现代建筑）
```

**不适合的图片**：
```
❌ 真人照片（会被拒绝）
❌ 暴力血腥内容
❌ 色情内容
❌ 版权受保护的内容
❌ 名人肖像
```

## 3️⃣ 📁 管理视频资产

### 功能亮点
✨ 统一管理所有视频  
✨ 实时显示生成进度  
✨ 自动刷新状态  
✨ 一键播放下载  

### 操作步骤

```
第1步：进入资产页（三种方式）
      方式A：点击底部"资产"Tab
      方式B：工作室 → "📁 管理视频资产"
      方式C：视频生成完成后自动跳转

第2步：查看视频列表
      顶部显示：
      - 总视频数量
      - 筛选选项
      
      每个视频卡片显示：
      - 缩略图预览
      - 标题（来自提示词）
      - 时长标签
      - 状态图标
      - 创建时间

第3步：查看不同状态的视频
      
      🟡 生成中：
      - 显示旋转的加载图标
      - 实时更新进度百分比
      - 中间显示进度条
      
      🟢 已完成：
      - 显示绿色勾图标
      - 鼠标悬停显示播放按钮
      - 点击即可播放
      
      🔴 失败：
      - 显示红色叉图标
      - 可以查看错误原因

第4步：自动刷新（无需操作）
      - 系统每5秒自动查询
      - 只刷新进行中的任务
      - 完成后自动更新状态

第5步：手动刷新
      - 点击右上角"刷新"按钮
      - 立即更新所有任务状态

第6步：播放视频
      - 点击已完成的视频卡片
      - 在新标签页打开播放
      - 可以下载保存

第7步：筛选和整理
      - 使用项目筛选下拉菜单
      - 按状态查看
      - 按时间排序
```

### 💡 资产管理技巧

**批量生成管理**：
```
1. 同时提交多个任务
   - 不同提示词
   - 不同模型
   
2. 在资产页统一查看
   - 所有任务一目了然
   - 各自独立进度
   
3. 等待全部完成
   - 自动更新状态
   - 完成后统一查看
   
4. 批量下载
   - 逐个点击下载
   - 或复制链接
```

**性能优化**：
```
✓ 定期清理失败的任务
✓ 下载后可删除本地记录
✓ 避免同时进行太多任务（建议3-5个）
✓ 使用项目分类管理
```

### 💡 专业创作工作流程

**标准工作流程**：
```
1️⃣ 构思阶段
   - 确定视频主题
   - 构思画面内容
   - 准备提示词草稿

2️⃣ 快速预览
   - 使用10秒模型
   - 测试提示词效果
   - 快速迭代调整

3️⃣ 优化提示词
   - 根据预览效果
   - 增加细节描述
   - 调整氛围表达

4️⃣ 正式生成
   - 使用15秒标准模型
   - 或Pro模型（重要项目）
   - 耐心等待完成

5️⃣ 后期管理
   - 在资产页查看
   - 下载保存
   - 分享发布
```

**批量创作流程**：
```
1️⃣ 准备阶段
   - 列出5-10个创意点子
   - 编写详细提示词
   - 分配不同模型

2️⃣ 批量提交
   - 依次创建任务
   - 同时进行3-5个
   - 避免过载

3️⃣ 进度监控
   - 在资产页查看
   - 关注失败任务
   - 及时重新提交

4️⃣ 统一管理
   - 全部完成后查看
   - 筛选满意作品
   - 批量下载

5️⃣ 作品发布
   - 整理分类

   - 社交媒体发布
