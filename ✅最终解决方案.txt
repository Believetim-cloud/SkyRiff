❌ 您遇到的问题
─────────────────────────────────────────────────────────────────

1. 想要前端按钮一键启动
   → ❌ 不可能：浏览器安全限制，禁止网页启动本地程序

2. 双击 .bat 或 .vbs 文件会变成TXT编辑
   → ❌ 文件关联设置问题

3. Figma Make 无法创建 .bat/.vbs 文件
   → ❌ 系统限制，会自动添加 .tsx 扩展名


✅ 我的解决方案
─────────────────────────────────────────────────────────────────

提供 3 种 **不需要双击文件** 的启动方法！

🚀 方法1：Win+R 直接运行（最推荐！）
═══════════════════════════════════════════════════════════════

优点：
  ✅ 不需要创建任何文件
  ✅ 不需要双击
  ✅ 4 步搞定
  ✅ 最简单快速


步骤：
─────────────────────────────────────────────────────────────────

1. 按键盘 Win+R 键

2. 输入（记得改路径）：

   cmd /c "cd /d C:\Users\k\Desktop\skyriff\server && npm install && npm start"
   
   ⚠️ 把 C:\Users\k\Desktop\skyriff 改成您项目的实际路径！

3. 按回车

4. 等待 "✅ 准备就绪！"

5. 刷新浏览器

6. 开始使用！


如何找到路径：
  1. 打开 server 文件夹
  2. 点击地址栏（会自动选中）
  3. Ctrl+C 复制
  4. 粘贴到命令中

💻 方法2：VS Code 终端（开发者推荐）
═══════════════════════════════════════════════════════════════

优点：
  ✅ 开发者最爱
  ✅ 简单快速
  ✅ 可以看到详细日志
  ✅ 最专业


步骤：
─────────────────────────────────────────────────────────────────

1. 用 VS Code 打开项目文件夹
   （右键项目文件夹 → "通过 Code 打开"）

2. 按 Ctrl+` 打开终端
   （或菜单：终端 → 新建终端）

3. 输入：

   cd server && npm install && npm start

4. 按回车

5. 等待 "✅ 准备就绪！"

6. 刷新浏览器

7. 完成！

📝 方法3：手动创建 .bat 文件
═══════════════════════════════════════════════════════════════

优点：
  ✅ 一劳永逸
  ✅ 创建一次，以后右键运行
  ✅ 不需要每次输入命令


步骤：
─────────────────────────────────────────────────────────────────

1. Win+R，输入 notepad，回车（打开记事本）

2. 复制粘贴这些代码：

┌─────────────────────────────────────────────────────────────┐
│ @echo off                                                   │
│ chcp 65001 >nul                                            │
│ cd /d "%~dp0"                                              │
│ echo.                                                       │
│ echo ========================================                │
│ echo    SkyRiff 服务器启动中...                             │
│ echo ========================================                │
│ echo.                                                       │
│ if not exist "node_modules" (                              │
│     echo [安装] 首次运行，正在安装依赖...                   │
│     call npm install                                        │
│ )                                                           │
│ echo [启动] 正在启动服务器...                              │
│ npm start                                                   │
│ pause                                                       │
└─────────────────────────────────────────────────────────────┘

3. 点击"文件" → "另存为"

4. 文件名输入：启动服务器.bat
   ⚠️ 注意：.bat 是扩展名

5. 保存类型：选择 **"所有文件 (*.*)"**
   ⚠️ 这个很重要！必须选"所有文件"！

6. 保存位置：选择项目的 server 文件夹

7. 点击"保存"

8. 找到保存的文件

9. 右键点击 → 选择 **"运行"**
   ⚠️ 不要双击！用右键运行！

10. 如果没有"运行"选项：
    右键 → "打开方式" → "命令提示符"

❓ 常见问题
─────────────────────────────────────────────────────────────────

Q：为什么不能在前端放个按钮一键启动？
A：浏览器安全限制，所有浏览器都禁止网页启动本地程序。
   这是为了保护用户安全，无法绕过。

Q：为什么双击 .bat 会变成TXT？
A：您电脑上的文件关联设置问题。
   解决：用右键"运行"，或者用方法1/2，不需要双击。

Q：我不知道项目路径怎么办？
A：打开 server 文件夹，点击地址栏，复制路径即可。

Q：首次运行很慢？
A：正常！首次需要下载依赖包，需要1-2分钟。
   下次启动就快了。

Q：端口被占用怎么办？
A：在命令行窗口按 Ctrl+C 停止服务器，重新启动。
