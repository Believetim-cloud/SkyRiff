📁 详细位置说明
─────────────────────────────────────────────────────────────────

1. 打开您的项目文件夹
   （可能在桌面、文档、或其他位置）

2. 进入 server 文件夹
   （项目文件夹里的一个子文件夹）

3. 找到 start.bat 文件
   （Windows批处理文件，有个齿轮图标）

4. 双击它！


文件夹结构：
┌─────────────────────────────────────┐
│ 项目文件夹/                          │
│   ├─ package.json                   │
│   ├─ src/                           │
│   ├─ server/                ← 进入这里│
│   │    ├─ package.json              │
│   │    ├─ real-api-server.js        │
│   │    └─ start.bat         ← 双击它！│
│   └─ ...                            │
└─────────────────────────────────────┘

🌐 接下来做什么
─────────────────────────────────────────────────────────────────

1. 不要关闭黑色窗口
   让它一直运行在后台

2. 回到浏览器

3. 按 F5 刷新页面

4. 启动助手会显示 "✅ 服务器在线"

5. 点击 "我稍后启动" 关闭提示

6. 点击底部 "工作室" 按钮

7. 开始生成视频！🎬

⚠️ 重要提示
─────────────────────────────────────────────────────────────────

✓ 服务器运行时不要关闭黑色窗口

✓ 如果要停止服务器：
  在黑色窗口中按 Ctrl+C

✓ 下次启动：
  再次双击 start.bat 即可

🔧 还有问题？
─────────────────────────────────────────────────────────────────

如果找不到 start.bat 文件：
  • 确认您在项目根目录
  • 进入 server 子文件夹
  • 如果还是找不到，刷新浏览器预览

如果双击没反应：
  • 可能需要先安装 Node.js
  • 访问：https://nodejs.org/
  • 下载并安装 LTS 版本
  • 重启电脑
  • 再次尝试

如果启动失败：
  • 查看错误信息
  • 截图发给我
  • 我会帮您解决
