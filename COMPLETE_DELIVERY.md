## 🎬 功能详解

### 功能1: 🎬 生成AI视频

**能做什么**:
- 输入文字描述，AI自动生成视频
- 无需任何素材，纯AI创作
- 支持8种专业模型
- 3-30分钟生成完成

**如何使用**:
```
1. 点击"工作室" → "生成AI视频"
2. 输入提示词（例如：可爱的狗 开飞机）
3. 选择模型（推荐：竖屏 15秒）
4. 点击"开始生成"
5. 等待3-5分钟
6. 完成！
```

**示例提示词**:
```
✓ "璀璨的星空夜景，流星划过天际，银河清晰可见"
✓ "金色日落下的海浪拍打沙滩，海鸥飞过"
✓ "繁华都市的霓虹灯光，车流穿梭，充满未来感"
✓ "森林中的小溪流水，阳光透过树叶，宁静祥和"
```

### 功能2: 📸 图片转视频

**能做什么**:
- 上传静态图片，让画面动起来
- 添加动态效果和特效
- 风格转换和画面增强
- 支持本地上传和URL输入

**如何使用**:
```
1. 点击"工作室" → "图片转视频"
2. 上传图片或输入URL
3. 输入提示词（例如：让画面动起来）
4. 选择模型
5. 开始生成
6. 完成！
```

**提示词技巧**:
```
添加效果：
✓ "让画面动起来"
✓ "添加飘落的樱花"
✓ "添加粒子光效"

风格转换：
✓ "改成电影风格"
✓ "变成动画风格"
✓ "转换为水彩画风格"
```

### 功能3: 📁 管理视频资产

**能做什么**:
- 查看所有生成的视频
- 实时显示生成进度
- 自动刷新状态（每5秒）
- 播放、下载视频

**如何使用**:
```
1. 点击底部"资产"Tab
   或
   点击"工作室" → "管理视频资产"

2. 查看视频列表：
   - 🟡 生成中：显示进度条
   - 🟢 已完成：可以播放
   - 🔴 失败：显示错误

3. 点击视频播放
4. 右上角"刷新"手动更新
```

**核心特性**:
```
✓ 自动轮询更新（每5秒）
✓ 实时进度显示
✓ 本地数据持久化
✓ 一键播放下载
✓ 项目分类管理
```
