## ⚡ 最常用的操作

### 启动开发环境

```bash
# Windows 用户
双击运行: 启动.bat

# Mac/Linux 用户
chmod +x 启动.sh
./启动.sh
```

### 遇到问题？

1. 先看 **02-常见问题FAQ.md**
2. 还是不行？检查后端是否启动成功
3. 确保前端能访问 http://localhost:8000
