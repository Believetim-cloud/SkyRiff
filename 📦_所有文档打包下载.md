## 💾 如何下载文档

### 方式1：在Figma Make中直接查看

在左侧文件树中，点击任意文档即可查看内容：

```
📁 /
  📄 TESTING_GUIDE.md        ← 点击查看
  📄 PHASE1_COMPLETE.md      ← 点击查看
  📄 PHASE2_COMPLETE.md      ← 点击查看
  📁 docs/
    📄 01-产品需求文档-PRD.md  ← 点击查看
    ...
```

### 方式2：下载整个项目

点击 Figma Make 左上角的菜单：
- **Export** → **Download ZIP**

下载后解压，所有文档都在里面。

### 方式3：复制粘贴（单个文档）

1. 点击文档打开
2. 全选（Ctrl+A / Cmd+A）
3. 复制（Ctrl+C / Cmd+C）
4. 粘贴到本地文本编辑器（VS Code / Sublime Text）
5. 保存为 `.md` 文件

### 方式4：使用Git（如果项目在Git仓库）

```bash
# 克隆项目
git clone <项目地址>

# 进入项目目录
cd skyriff

# 所有文档都在 /docs 和根目录
ls -la docs/
ls -la *.md
```
