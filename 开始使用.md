## 🎬 3步开始

### 步骤1️⃣：启动服务器

```bash
cd server
npm install
npm start
```

看到这个就成功了：
```
✅ 准备就绪！可以开始使用真实API
```

### 步骤2️⃣：打开前端

点击 **"Open in New Tab"** 按钮

### 步骤3️⃣：生成视频

```
1. 点击底部"工作室"（紫色大圆按钮）
2. 点击"🎬 生成AI视频"
3. 输入："可爱的狗 开飞机"
4. 选择："竖屏 15秒"
5. 点击"开始生成"
6. 等待3-5分钟
7. 自动跳转到"资产"页
8. 点击视频播放 ▶️
```

## 🐛 遇到问题？

### 界面显示不正常
```
→ 刷新浏览器页面
→ 使用Chrome/Safari
```

### 视频生成失败
```
→ 查看服务器终端日志
→ 检查网络连接
→ 确认API Key有效
```

### 看不到视频
```
→ 点击资产页"刷新"
→ 等待生成完成（3-5分钟）
→ 检查视频状态是否为"已完成"
```

### 服务器启动失败
```bash
# 端口被占用
lsof -ti:3001 | xargs kill -9

# 重新安装依赖
rm -rf node_modules
npm install
```
