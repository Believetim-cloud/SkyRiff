# 🚀 SkyRiff API 快速开始

## 10分钟完成配置并生成第一个视频

---

## 步骤1: 配置API Key（2分钟）

### 打开配置文件

找到并打开：`/src/app/services/api-config.ts`

### 替换API Key

```typescript
export const API_CONFIG = {
  BASE_URL: 'http://prod-cn.your-api-server.com',
  
  // 👇 替换这里的API Key
  API_KEY: 'YOUR_API_KEY', // 改成你的真实API Key
  
  // ...其他配置
};
```

> ⚠️ **重要**: 确保API Key是有效的，否则所有请求都会失败！

---

## 步骤2: 测试连接（3分钟）

### 打开浏览器控制台

按 `F12` 或右键选择"检查"

### 在Console中测试

```javascript
// 测试API连接
fetch('http://prod-cn.your-api-server.com/v1/videos', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: '测试视频',
    model: 'sora2-portrait'
  })
})
.then(res => res.json())
.then(data => console.log('成功!', data))
.catch(err => console.error('失败:', err));
```

如果看到返回的任务ID，说明连接成功！

## 🐛 遇到问题？

### API Key错误
```
错误: 401 Unauthorized
解决: 检查 api-config.ts 中的 API_KEY 是否正确
```

### 视频生成失败
```
错误: 任务状态变为 failed
解决: 
  1. 检查提示词是否合规
  2. 避免真人图像
  3. 避免暴力、色情等内容
```

### 进度不更新
```
问题: 进度一直显示0%
解决: 
  1. 等待1-2分钟（API需要处理时间）
  2. 点击"刷新"按钮
  3. 检查浏览器控制台错误
```

### 视频无法播放
```
问题: 点击播放没反应
解决: 
  1. 确认视频已完成（绿色勾）
  2. 右键复制视频链接，在新标签页打开
  3. 检查网络连接
```
