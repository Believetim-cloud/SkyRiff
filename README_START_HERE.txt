🔧 如果遇到问题
================================================================================

问题1：安装失败或很慢
  → 确保有网络连接
  → 使用国内镜像（已默认使用清华源）

问题2：无法启动后端
  → 运行：backend\🔍检查环境.bat
  → 查看详细错误信息

问题3：端口被占用
  → 关闭其他占用8000端口的程序
  → 或修改启动端口
