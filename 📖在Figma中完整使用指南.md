## 🚀 完整使用流程

### 1️⃣ 首次进入

当您在 Figma Make 中打开 SkyRiff 时：

```
📱 显示欢迎页面
   ↓
🔍 检测服务器状态（3秒超时）
   ↓
🔄 自动启用离线模式
   ↓
💡 显示"演示模式"提示
```

**操作**：
- 点击"**开始使用**"按钮
- 或者点击底部的"**工作室**"按钮

---

### 2️⃣ 生成 AI 视频

进入视频工作室后：

1. **选择"生成 AI 视频"标签页**
   
2. **输入创意描述**，例如：
   ```
   一只可爱的小猫在花园里追逐蝴蝶
   ```
   ```
   未来城市的霓虹夜景，赛博朋克风格
   ```
   ```
   海浪拍打沙滩，阳光洒在海面上
   ```

3. **选择模型和参数**：
   - 模型：标准版或 Pro 版
   - 方向：竖屏（Portrait）或横屏（Landscape）
   - 时长：5s / 10s / 15s / 25s

4. **点击"生成视频"按钮**

5. **观看实时进度**：
   - 标准模型：约 10 秒完成
   - Pro 模型：约 20 秒完成
   - 进度条实时更新

6. **视频生成完成**：
   - ✅ 显示"生成成功"提示
   - 🎬 自动显示视频播放器
   - 💾 视频已保存到资产页面

---

### 3️⃣ 图片转视频

1. **切换到"图片转视频"标签页**

2. **上传图片**：
   - 点击上传区域选择本地图片
   - 或者输入图片 URL

3. **添加提示词**，描述想要的动画效果：
   ```
   让这张图片中的云朵动起来
   ```
   ```
   添加下雨的效果
   ```

4. **选择模型和参数**

5. **点击"开始转换"**

6. **等待生成完成**

---

### 4️⃣ 查看视频资产

1. **点击底部导航栏的"资产"**

2. **查看所有生成的视频**：
   - 📊 实时显示生成进度
   - ✅ 已完成的视频可以播放
   - ⏱️ 显示生成时间
   - 🎯 显示视频时长

3. **播放视频**：
   - 将鼠标悬停在视频缩略图上
   - 点击播放按钮
   - 在新标签页中播放（或使用内置播放器）

4. **自动刷新**：
   - 每 5 秒自动检查进行中的视频
   - 或者点击"刷新"按钮手动刷新

## 🔧 技术原理

### 离线模式工作流程

```
用户点击"生成视频"
    ↓
前端创建虚拟任务 ID
    ↓
启动进度模拟计时器（每 0.5秒更新）
    ↓
进度: 0% → 25% → 50% → 75% → 100%
    ↓
随机分配一个真实可播放的视频 URL
    ↓
保存到 localStorage
    ↓
显示在资产页面
    ↓
可以播放和下载
```

### 数据存储

- **存储位置**：浏览器 localStorage
- **存储内容**：
  - 任务 ID
  - 提示词
  - 视频 URL
  - 生成时间
  - 模型信息
  - 进度状态

- **持久化**：
  - 关闭浏览器后数据仍然保留
  - 刷新页面不会丢失
  - 支持导出和备份

## 💬 常见问题

### Q: 为什么显示"演示模式"？

**A:** 因为后端服务器未启动，系统自动切换到离线模式。在 Figma Make 的在线预览环境中，这是正常且推荐的使用方式。

### Q: 生成的视频是真实的吗？

**A:** 是的！所有视频都是真实可播放的 MP4 文件，来自 Google 官方测试视频库。虽然内容不是根据您的提示词生成的，但可以完整体验整个流程。

### Q: 视频会永久保存吗？

**A:** 视频链接和元数据保存在浏览器的 localStorage 中。只要不清除浏览器数据，就会一直保留。

### Q: 可以下载视频吗？

**A:** 可以！点击视频播放器中的下载按钮，或者右键点击视频选择"保存视频"。

### Q: 如何使用真实 API？

**A:** 如果您需要使用真实的 Sora API 生成自定义视频，需要：
1. 下载项目到本地
2. 配置 API 密钥
3. 启动后端服务器
4. 详见 `🚀快速启动指南.md`
